# 把服务器上所有分割好的文件下载到本地

cat sql.books.* > books.sql
# 把所有已经下载好的、分割好的文件，再次合成同一个文件
md5sum books.sql
# 在本地生成的hash值和服务器上生成的hash值对比一下，如果完全一样，备份结束

命名规则（PEP8）

  • 大写字母开头的：一般是类
  • 小写字母开头的：一般是方法或者变量或者库、模块
  • 方法尽量用蛇型命名法： get_books_infos()
